# --------------------
#  AGDAC_Automatic Gas Data Acquisition for Gas Counters 
# --------------------

An innovative Market-Oriented, Open-Source Multiplexing System for 
advanced Biogas Measurements in Research Experiments.


Folder description:

AGDAC_AR: Arduino IDE code (code 1) that sense and record tilts.

AGDAC_PY: PYTHON code (code 2) that records and 
saves Gas Volume production data in CSV file.
